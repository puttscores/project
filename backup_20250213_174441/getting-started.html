Getting Started with Workbird

Follow these simple steps to start trading smarter

1

Get Your API Keys

OpenRouter API Key

  1. Visit OpenRouter.ai
  2. Create an account or sign in
  3. Navigate to the API section
  4. Generate a new API key
  5. Copy and save your API key securely

Alpaca Trading Account

  1. Go to Alpaca Markets
  2. Sign up for a paper trading account
  3. Once registered, go to your Dashboard
  4. Find your API Keys section
  5. Copy both your API Key ID and Secret Key
2

Install the Extension

  1. Download the Workbird extension package
  2. Unzip the downloaded file
  3. Open Chrome and go to chrome://extensions/
  4. Enable "Developer mode" in the top right
  5. Click "Load unpacked" and select the unzipped folder
  6. The Workbird icon should appear in your Chrome toolbar
3

Configure the Extension

  1. Click the Workbird icon in your Chrome toolbar
  2. Click the settings icon (⚙️)
  3. Enter your OpenRouter API key
  4. Enter your Alpaca API Key ID and Secret Key
  5. Save your settings
4

Start Trading

  1. Open your preferred trading chart (TradingView, Yahoo Finance, etc.)
  2. Click the Workbird icon to open the extension
  3. Click "Start Trading" to begin analysis
  4. Wait for trading opportunities to appear
  5. Review and confirm trades before execution

Important: Start with paper trading to familiarize yourself with the system. Real trading involves substantial risk of loss.
