Frequently Asked Questions

Everything you need to know about Workbird

Which charting platforms are supported?

Workbird works with any real-time chart platform accessible through your browser. This includes:

  • TradingView (Free or Premium)
  • Yahoo Finance Charts
  • Alpaca Markets Charts
  • Any other web-based charting platform

Simply open your preferred charting platform in Chrome, and Workbird will automatically start analyzing the price action in real-time.

What API keys do I need?

You'll need two API keys to get started:

  • OpenRouter API key for AI analysis
  • Alpaca Markets API key for trade execution

Both platforms offer free tiers that are perfect for getting started. Check our Getting Started guide for detailed setup instructions.

How does the AI analysis work?

Workbird uses OpenRouter's state-of-the-art language models to:

  • Analyze real-time price action and patterns
  • Evaluate technical indicators
  • Identify potential trading opportunities
  • Calculate optimal entry and exit points

The AI continuously learns and adapts to market conditions while following strict risk management rules.

Is paper trading supported?

Yes! We strongly recommend starting with paper trading. Simply use your Alpaca paper trading API keys instead of live trading keys. This allows you to:

  • Practice with real market data
  • Test strategies risk-free
  • Understand the platform's features
  • Build confidence before live trading

Can I customize the trading logic?

Absolutely! As an open-source project, Workbird is fully customizable:

  • Modify risk management parameters
  • Add custom technical indicators
  • Implement your own trading strategies
  • Contribute improvements to the community

What are the system requirements?

Workbird is lightweight and runs in Chrome:

  • Google Chrome browser
  • Internet connection for real-time data
  • API keys for OpenRouter and Alpaca
  • No additional software needed
